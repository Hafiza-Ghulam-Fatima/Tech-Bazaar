Tech Bazaar is a full-stack e-commerce web application for electronic products, developed as a **term project for the Web Technologies course**.  
The system allows users to browse products, manage a shopping cart, place orders, and securely authenticate, while admins can manage products and orders through a dedicated dashboard.

The project is built using the **PERN stack** (PostgreSQL, Express.js, React.js, Node.js) and follows RESTful architecture with JWT-based authentication.

**Key Features**

**User Operations**
- Register and log in with secure JWT authentication
- Browse electronic products with search and filtering
- Add, update, or remove products from shopping cart
- Place orders and view order history
- Responsive interface for desktop and mobile devices

**Admin Operations**
- Admin login with authorization
- Add, update, and delete products
- Manage customer orders
- Access admin dashboard for management tasks

**Tech Stack**

**Frontend**
- React.js
- Context API (state management)
- HTML & CSS

**Backend**
- Node.js
- Express.js (RESTful APIs)
- JWT Authentication
- bcrypt (password hashing)

**Database**
- PostgreSQL
- Normalized relational database schema

**Concepts Used**
- RESTful API design
- Authentication & Authorization using JWT
- State management with Context API
- Database design and normalization
- Secure password handling
- Full-stack integration

**How to Run the Project**

1. Clone the repository
2. Install dependencies for backend and frontend
3. Configure PostgreSQL database
4. Run backend server
5. Start frontend application

**System Requirements**
- Node.js
- PostgreSQL
- Web browser
- Basic understanding of PERN stack
